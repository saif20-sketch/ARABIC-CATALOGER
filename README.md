# 🏥 Medical AI Librarian - Enhanced Edition

**Intelligent Medical Book Cataloging with Advanced NLM Classification**

## ✨ الميزات
- تصنيف NLM متقدم بقاعدة بيانات شاملة
- استرجاع أغلفة الكتب من 6 مصادر مختلفة
- استخراج البيانات الوصفية من 4+ واجهات برمجة
- توليد سجلات MARC21 كاملة مع تنسيقات ISO 2709 و MARCXML و JSON
- لوحة إحصاءات تحليلية
- دعم الذكاء الاصطناعي (نماذج مجانية)

## 🚀 التشغيل السريع
1. ثبّت المتطلبات:  
   `pip install -r requirements.txt`
2. أنشئ ملف `.env` وأضف `HF_TOKEN` الخاص بك.  
3. شغّل التطبيق:  
   `python Run20Application.py`

## 📦 المتطلبات
- Python 3.8+
- راجع `requirements.txt`
